PART 2: TOMER HEN'S ADVOCATE FUNNEL STRATEGIES FOR THE QUIZ WEBSITE
--------------------------------------------------------------------------------
Tomer Hen's marketing philosophy focuses on moving away from transactional ad-spend
to build a self-sustaining organic flywheel through micro-advocacy and relationship seeding.

1. Educator & Organic Advocate Kits (Upfront Seeding)
   * Strategy: Create a new static page or modal path: `/seeding` or `/educators`.
   * Action: Offer local garden club leaders, science teachers, and environmental educators
     a free "Compost Educator Package" (containing physical print-ready waste sorting guides,
     classroom posters with QR codes pointing to the web teaser, and custom lesson prompts).
   * Hen's Rationale: Providing massive, un-gated value with zero sales pitches turns these
     highly-visible community educators into active advocates who print and distribute the app.

2. Micro-Influencer Direct Seeding Playbook
   * Strategy: Partner with micro-creators (urban homesteaders and eco-bloggers with 5k-30k followers)
     using value rather than paid transactions.
   * Action: Send them a personalized email offering an exclusive unlock coupon code
     specifically named after their channel (e.g., `CREATORNAME99` or `ORGANICSOIL99`).
   * Action: Provide them with custom visual badges (like "Balcony Worm Whisperer") and scoreboards
     they can showcase on social channels. This gives their followers a tangible reason to play
     the web teaser, download the APK, and use the creator's code in Settings.

3. The "Share progress" Viral Flywheel
   * Strategy: Build sharing loops that naturally redirect back to the acquisition funnel.
   * Action: Add an "Invite a Composting Buddy" referral block on the landing page footer
     and post-teaser screens. This button generates a pre-formatted invite link to direct-message:
     "Hey! I'm testing my composting skills in the Kitchen Scraps teaser. Try it out: kitchen-scraps.web.app"

PART 3: IMPROVEMENT PLAN FOR QUIZ APP (F:\Dev\Projects\kitchen_scraps_quiz)
--------------------------------------------------------------------------------
After reviewing the Flutter quiz app screens, models, and providers:

1. settings_screen.dart (Redemption Flow Enhancement)
   * Current State: Entering "GARDENGOLD99" plays standard confetti and opens a dialog.
     The theme provider swaps the theme, but it lacks a premium feel.
   * Improvement: Build a dedicated custom transition overlay. When the code is successfully
     validated, display a full-screen gold particle burst/sweep effect. Smoothly transition
     the color scheme to the Amber Gold Garden palette using a radial background animation
     instead of a flat dialog popup to increase the psychological value of the skin.

2. quiz_screen.dart & quiz_state.dart (Urgency in Streak-Saves)
   * Current State: The streak-save dialog checks if the user is about to lose a streak
     and prompts them to watch a rewarded ad to keep progress alive.
   * Improvement: Add an active visual countdown timer (e.g., 5 seconds) to the Streak-Save
     dialog card. A countdown adds psychological urgency (loss aversion) that prompts players
     to tap "Read & Save Streak" immediately rather than abandoning the app.

3. quiz_summary_screen.dart (Optimize Web Teaser Funnel for Desktop Players)
   * Current State: The web teaser summary card displays text links and an "Install Android App"
     button which redirects to the download page.
   * Improvement: When played on a desktop browser, a user cannot directly install the APK.
     Add a dynamically generated QR code directly on the web summary card. A desktop player
     can scan the QR code with their mobile phone to instantly pull up the APK download page
     and initiate transfer, bypassing device-sync friction.
   * Improvement: Provide a copy button on the teaser summary card specifically to copy
     their web teaser scorecard directly to clipboard as formatted text for sharing.

4. download_screen.dart (Verify Direct Access)
   * Current State: The download_screen.dart is built into the Flutter app shell.
   * Improvement: Ensure the download instructions screen is highly visual, displaying
     animated guides for bypassing the "Unknown Sources" warning prompt, identical to the
     landing page web instructions.
